# 🚀 INÍCIO RÁPIDO - Nvision Informatica

## ⚡ 3 PASSOS PARA COMEÇAR

### 1️⃣ WINDOWS - Duplo Clique
```
run.bat
```
**É isso!** O script vai instalar tudo e abrir no navegador automaticamente.

### 2️⃣ MAC / LINUX - Terminal
```bash
chmod +x run.sh
./run.sh
```

### 3️⃣ ABRA NO NAVEGADOR
```
http://localhost:5000
```

---

## 📋 O QUE VOCÊ VÊ

### 🏠 Dashboard
- Resumo de atendimentos
- Formulário para abrir novos chamados
- Últimos chamados registrados

### 📑 Chamados (Listagem)
- Listar todos os chamados
- Filtrar por status ou cliente
- Ver detalhes e atualizar status

### 📊 Relatórios
- Selecionar mês/ano
- Ver estatísticas por cliente
- Ver distribuição por categoria
- **Baixar PDF** para apresentar aos clientes

---

## 🎯 COMO USAR

### ➕ Abrir Novo Chamado
1. Clique em "Dashboard"
2. Preencha:
   - **Cliente** (nome da empresa)
   - **Categoria** (tipo de atendimento)
   - **Data/Hora** (quando atendeu)
   - **Descrição** (detalhes do problema)
3. Clique em "Abrir Chamado"

### 📈 Gerar Relatório em PDF
1. Vá para "Relatórios"
2. Selecione mês e ano
3. Clique em "Baixar PDF"
4. Arquivo pronto para enviar aos clientes!

## ❓ PRECISA DE AJUDA?

Se tiver erro ao executar:

### Windows
1. Abra PowerShell como Administrador
2. Execute: `python -m pip install -r requirements.txt`
3. Execute: `python run.py`

### Mac/Linux
```bash
python3 -m pip install -r requirements.txt
python3 run.py
```
