## ⚡ SUPER RESUMIDO (30 segundos)

```bash
# 1. Dentro da pasta do projeto, configure Git:
git init
git add .
git commit -m "Deploy NVision"
git branch -M main
git remote add origin https://github.com/seu-usuario/seu-repo.git
git push -u origin main

# 2. Acesse render.com
# 3. Conecte com GitHub
# 4. Clique "New Web Service"
# 5. Selecione seu repositório
# 6. Clique "Create"

# Pronto! Em 5 minutos terá URL como: https://seu-app.onrender.com
```
